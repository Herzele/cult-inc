# Cult Inc — Balance Sheet

Working document. Update whenever a formula or constant changes. Always record *why* a value was chosen.

---

## Resources

| Resource | Unit | Description |
|----------|------|-------------|
| Belief | B | Primary currency. Generated by followers, clicks, rituals. |
| Followers | F | Population. Generate Belief passively. Capped by Shelter. |
| Shelter | S | Housing capacity. Increases via upgrades. Starts at 10. |
| Heat | H | 0–100. Rises with Follower count and events. High Heat triggers crackdowns. |
| Doctrine | D | Prestige currency. Earned per Revelation. Spent on permanent upgrades. |

---

## Core Formulas

### Belief Generation
```
belief_per_second = followers_preaching * belief_per_follower * global_multiplier
```
- `belief_per_follower` starts at **1.0 B/s**
- `global_multiplier` starts at **1.0**, improved by upgrades

### Follower Recruitment Cost
```
recruit_cost(n) = base_cost * growth_rate ^ n
```
- `base_cost` = **10 B** for first follower
- `growth_rate` = **1.12** (roughly doubles every 6 purchases)
- Softcap: cost increase slows above 50 followers (growth_rate → 1.07)

### Heat Generation
```
heat_per_minute = (followers / 10) + active_event_heat
```
- Heat decays at `heat_decay_rate = 0.5 / minute` when below 30
- Heat decays at `0.1 / minute` when 30–70
- Heat does NOT decay above 70 (requires active Secrecy upgrades)

### Offline Progress
```
offline_belief = min(belief_per_second * offline_seconds, max_offline_cap)
max_offline_cap = belief_per_second * 14400  // 4 hours cap
```

## Heat Thresholds & Events

| Heat | Event |
|------|-------|
| 25 | Local newspaper article — +2 Heat, −5 Followers (desertion) |
| 50 | Police visit — pause recruiting for 60s |
| 75 | Government inquiry — Heat locked at 75 until Secrecy upgrade purchased |
| 100 | Crackdown — lose 30% Followers, Heat reset to 40 |

---

## Tuning Notes

- Target for first follower purchase: **~30 seconds** of idle clicking
- Target for first upgrade: **~3 minutes**
- Players should feel "oops too much Heat" for the first time around **minute 10–15**
- Prestige should feel achievable in **2–4 hours** of first run, faster on repeat
